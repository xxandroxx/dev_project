1.- El sitio web funciona con "localstorage" por ende si hay más de dos datos almacenados en el local storage no se abrirál el pop up del formulario 
	en la página de inicio.
2.- La selección de personajes es solamente una página para elegir un personaje pero no se almacena a la hora de seleccinar uno.
3.- En la página de "usuario" se muestra el "nombre de usuario" y "el nombre del jugador", no se muestra la contraseña.
4.- Para poder ver los datos en la página "usuario" se tiene que rellenar el formulario que aparece como pop up en la página de inicio.
5.- Al apretar el enlace "cerrar sesión" se borrarán los datos puestos en el localstorage.
6.- Al cerrar sesión, se puede ir a la página "inicio" y volverá a aparecer el pop up del formulario.
7.- El sitio web contiene sonidos en la página de "selección de personaje", asi que hay que tener el audio del computador encendido.
